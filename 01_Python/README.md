# 🐍 Module 1 - Python Fundamentals

## Lesson 1 - Python Memory Model

### 🎯 Objective

Understand how Python stores data in memory and how variables interact with objects.

---

## 📚 Topics Covered

- RAM
- Objects
- Variables
- References
- id()
- Value, Type & Identity

---

## 🧠 Key Learnings

- Everything in Python is an object.
- Variables refer to objects.
- Variables do not store values.
- Multiple variables can refer to the same object.
- One variable can refer to only one object at a time.
- Every object has:
  - Value
  - Type
  - Identity
- `id()` returns the identity of an object.
- Assignment changes the reference, not the object.
